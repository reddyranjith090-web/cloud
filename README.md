# Java Swing Calculator

## Overview

This is a simple calculator application developed using Java Swing. It provides a graphical user interface (GUI) that allows users to perform basic arithmetic operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Input validation for numeric values
* Division-by-zero error handling
* Simple and user-friendly interface

## Technologies Used

* Java
* Java Swing
* AWT Event Handling

## Requirements

* Java Development Kit (JDK) 8 or later

## How to Run

1. Save the source code as `Calculator.java`.
2. Open a terminal or command prompt.
3. Compile the program:

   ```
   javac Calculator.java
   ```
4. Run the application:

   ```
   java Calculator
   ```

## How to Use

1. Enter the first number.
2. Enter the second number.
3. Click one of the operation buttons:

   * **Add**
   * **Subtract**
   * **Multiply**
   * **Divide**
4. The result will be displayed in the **Result** field.

## Error Handling

* Displays an error message if non-numeric input is entered.
* Prevents division by zero and shows an appropriate warning.
